# Emotion Sentiment Analysis using NLP

An NLP-based Emotion Sentiment Analysis web application that predicts human emotions (**Joy, Anger, and Fear**) from text using **TF-IDF** and **Logistic Regression**. The project includes text preprocessing, feature extraction, model training, and an interactive Streamlit web interface.

### 🚀 Live Demo
https://codealphasentimentanalysis-n5w2wngeubpp3jmm7wuv3n.streamlit.app/

### 🛠️ Tech Stack
- Python
- Scikit-learn
- NLTK
- TF-IDF Vectorizer
- Logistic Regression
- Streamlit

### 📊 Model Performance
- Validation Accuracy: **94.36%**
